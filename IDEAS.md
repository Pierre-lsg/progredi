# 💡 Idées d'évolutions pour Progredi

Ce document recense les fonctionnalités et améliorations envisagées pour les futures versions de l'application.

## 📊 Métriques et Statistiques
- [ ] **Tableau de bord avancé** : Graphiques détaillés avec évolution sur plusieurs jours/semaines.
- [ ] **Statistiques par catégorie** : Taux de réussite ventilé par catégorie.
- [ ] **Filtres de date** : Afficher les statistiques pour une période personnalisée.

## 💻 OS
- [ ] **Widget Android** : Afficher les objectifs du jour sur le bureau Android.
- [ ] **Widget iOS** : Afficher les objectifs du jour sur le bureau iOS.

## ⚙️ Technique et Synchronisation
- [ ] **Backend / Sync** : Synchroniser les données entre plusieurs appareils via un compte utilisateur.

## ✅ Réalisé récemment
- [x] **Langues** : Support multilingue (Français, Anglais, Espagnol, Allemand, Italien).
- [x] **Fréquences précises** : 
    - [x] Hebdomadaire : choix du jour de la semaine.
    - [x] Mensuel : choix du jour du mois (gestion du 31).
    - [x] Annuel : choix de la date dans l'année.
- [x] **Confirmation sensorielle** : Son et vibration lors de la validation.
- [x] **Saisie vocale** : Bouton microphone pour dicter le titre de l'objectif.
- [x] **Glisser-déposer** : Réorganiser manuellement l'ordre des objectifs.
- [x] **Mode Hors-ligne avancé** : Améliorer la gestion du cache PWA.
- [x] **Thèmes personnalisés** : Permettre de choisir sa couleur d'accentuation.
- [x] **Tri par catégorie** : Regrouper visuellement les objectifs par domaine (Santé, Maison, etc.).
- [x] **Priorités** : Ajouter un niveau d'importance (Basse, Moyenne, Haute) et trier en conséquence.
- [x] **Recherche textuelle** : Pour retrouver rapidement un objectif dans une longue liste.
- [x] **Favoris** : Épingler certains objectifs en haut de la liste.
- [x] **Vue Graphique** : Graphiques en barres ou en camembert montrant le taux de réussite par semaine/mois.
- [x] **Séries (Streaks)** : Afficher le nombre de jours consécutifs de réussite pour chaque objectif quotidien.
- [x] **Historique détaillé** : Calendrier permettant de cliquer sur un jour passé pour voir le détail des réalisations.
- [x] **Rappels personnalisés** : Définir une heure de notification par objectif.
- [x] **Notification de bilan** : Un rappel le soir si des objectifs ne sont pas encore remplis.
